- ### Requiere las siguientes variables de entorno (archivo: .env)
```sh
VITE_API_URL= api del backend url/api
VITE_SOCKET_URL= url del server
VITE_MAPS_KEY=
```

- ### Comandos para iniciar el proyecto

```sh
cd ecos_client
npm i
npm run dev
```
